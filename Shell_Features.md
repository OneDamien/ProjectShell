

####Piping####
/* Contributor Name: Matt*/

Enable your shell to connect the output of one program to the input of another.

ls -l | grep .pdf | less


####History####
/* Contributor Name: */

user$ cd
  <arrow up>
user$ ls


####Tab completion####
/* Contributor Name: */

Hitting in your shell will suggest commands that match your partial command.

user$ wg <tab>
user$ wget
